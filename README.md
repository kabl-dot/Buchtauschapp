# Buchtauschapp

## Beschreibung

Dieses Projekt entstand im Rahmen des Projektes Data-Mart-Erstellung in SQL.

Es handelt sich um eine relationale Datenbank für eine Buchtausch-App und ermöglicht die Verwaltung von:

- Benutzer:innen
- Rollen
- Büchern
- Autor:innen
- Verlagen
- Sprachen
- Genres
- Buchexemplaren
- Angeboten
- Standorten
- Ausleihtransaktionen
- Bewertungen
